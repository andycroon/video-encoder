**Step B: Create tests/test_db.py (7 specs, all RED)**

All tests use real SQLite with `tmp_path` pytest fixture and `asyncio.run()` — no mocking, consistent with Phase 1 philosophy.

Helper at top of test file:
```python
def default_config() -> dict:
    return {
        "vmaf_min": 96.2, "vmaf_max": 97.6,
        "crf_min": 16, "crf_max": 20, "crf_start": 17,
        "audio_codec": "eac3",
        "x264_params": {"preset": "slow", "maxrate": "12000K"}
    }
```

Write these 7 test functions:

**test_job_survives_restart(tmp_path):**
- `await init_db(db_path)`
- `job_id = await create_job(db_path, "/source/video.mkv", default_config())`
- Open fresh connection (simulate restart): `result = await get_job(db_path, job_id)`
- Assert result["status"] == "QUEUED"
- Assert result["source_path"] == "/source/video.mkv"
- Assert result["config"] is a dict (not a raw JSON string) with key "vmaf_min"

**test_wal_mode_active(tmp_path):**
- `await init_db(db_path)`
- Open a fresh aiosqlite connection (NOT via get_db — raw aiosqlite.connect)
- Execute `PRAGMA journal_mode` and fetchone
- Assert row[0] == "wal"

**test_stale_job_recovery(tmp_path):**
- `await init_db(db_path)`
- Create a job, manually UPDATE it to status=RUNNING with heartbeat_at set to 2 hours ago (ISO-8601 UTC string)
- `count = await recover_stale_jobs(db_path)`
- Assert count == 1
- `result = await get_job(db_path, job_id)`
- Assert result["status"] == "QUEUED"
- Assert result["heartbeat_at"] is None

**test_chunk_crud(tmp_path):**
- `await init_db(db_path)`
- Create a job, then `chunk_id = await create_chunk(db_path, job_id, chunk_index=0)`
- `await update_chunk(db_path, chunk_id, crf_used=17.0, vmaf_score=96.8, iterations=2, status="DONE")`
- `chunks = await get_chunks(db_path, job_id)`
- Assert len(chunks) == 1
- Assert chunks[0]["crf_used"] == 17.0
- Assert chunks[0]["vmaf_score"] == 96.8
- Assert chunks[0]["iterations"] == 2
- Assert chunks[0]["status"] == "DONE"

**test_step_crud(tmp_path):**
- `await init_db(db_path)`
- Create a job, then `step_id = await create_step(db_path, job_id, "FFV1 encode")`
- `await update_step(db_path, step_id, "DONE")`
- `steps = await get_steps(db_path, job_id)`
- Assert len(steps) == 1
- Assert steps[0]["step_name"] == "FFV1 encode"
- Assert steps[0]["status"] == "DONE"

**test_log_append(tmp_path):**
- `await init_db(db_path)`
- Create a job
- `await append_job_log(db_path, job_id, "Chunk 1: CRF 17 -> VMAF 96.8 PASS")`
- `await append_job_log(db_path, job_id, "Chunk 2: CRF 18 -> VMAF 97.1 PASS")`
- `result = await get_job(db_path, job_id)`
- Assert "Chunk 1" in result["log"]
- Assert "Chunk 2" in result["log"]
- Assert result["log"].index("Chunk 1") < result["log"].index("Chunk 2")

**test_config_roundtrip(tmp_path):**
- `await init_db(db_path)`
- cfg = default_config()
- `job_id = await create_job(db_path, "/source/video.mkv", cfg)`
- `result = await get_job(db_path, job_id)`
- Assert result["config"] == cfg (dict equality — get_job must deserialize JSON)
- Assert result["config"]["vmaf_min"] == 96.2
- Assert result["config"]["x264_params"]["maxrate"] == "12000K"
