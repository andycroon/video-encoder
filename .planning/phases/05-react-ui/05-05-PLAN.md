Design direction (from CONTEXT.md Claude's Discretion):
- Video encoder tool — lean into the "professional media tool" aesthetic
- Dark background is correct (neutral-950 base is right)
- Accent: a single strong accent color — recommend electric blue (#3b82f6 / blue-500) for primary actions and active states
- Secondary accent: a warm amber for RUNNING/active indicators (not generic blue-on-blue)
- Typography: Inter or system-ui for UI text; JetBrains Mono or ui-monospace for file paths, log text, CRF/VMAF numbers
- Status badge palette (from Claude's Discretion — design these):
  - QUEUED: slate-600 background, slate-300 text
  - RUNNING: blue-600 background, white text (prominent — this is the active state)
  - PAUSED: amber-700 background, amber-100 text
  - DONE: emerald-800 background, emerald-200 text
  - FAILED: red-800 background, red-200 text
  - CANCELLED: neutral-700 background, neutral-400 text
- Active stage indicator: replace plain "▶" with a pulsing dot or animated indicator
- Job row hover: subtle highlight (bg-white/3 or similar)
- Chunk table: alternating row shading (every-other-row bg-white/2)
- Log panel: true terminal feel — very dark background (bg-black or #0a0a0a), green-300 or neutral-400 text
