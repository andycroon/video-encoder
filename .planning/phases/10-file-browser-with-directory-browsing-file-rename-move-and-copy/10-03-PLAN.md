**2. Inline Rename (per D-14, D-15, D-16):**

Add state:
```typescript
const [renamingPath, setRenamingPath] = useState<string | null>(null);
const [renameValue, setRenameValue] = useState('');
```

`startRename(entry: BrowseEntry)`:
- Set `renamingPath = entry.path`
- Set `renameValue = entry.name` (filename without directory)
- Close context menu

In file row rendering, when `entry.path === renamingPath`, replace the filename span with:
```tsx
<input
  type="text"
  value={renameValue}
  onChange={(e) => setRenameValue(e.target.value)}
  onKeyDown={(e) => {
    if (e.key === 'Enter') confirmRename(entry);
    if (e.key === 'Escape') cancelRename();
  }}
  onBlur={() => cancelRename()}
  autoFocus
  style={{
    flex: 1,
    background: 'var(--bg)',
    border: '1px solid #4080ff',
    borderRadius: 3,
    padding: '2px 6px',
    fontSize: 13,
    color: 'var(--txt)',
    outline: 'none',
    fontFamily: 'inherit',
  }}
/>
```

`confirmRename(entry: BrowseEntry)`:
- If `renameValue.trim() === '' || renameValue === entry.name`, cancel
- Call `renameFile(entry.path, renameValue.trim())`
- On success, refresh the panel's directory listing by calling `browse(currentPath)` again
- On error (409 = already exists), show alert or inline error
- Reset `renamingPath` to null

`cancelRename()`: Set `renamingPath` to null.

**4. Add to Queue (per D-19):**

Import: `import { submitJob } from '../api/jobs';`
Import: `import { useJobsStore } from '../store/jobsStore';`

Inside the FileBrowser component, access store with individual selectors:

`handleAddToQueue(entry: BrowseEntry)`:
- Close context menu
- Find default profile: `const profile = profiles.find(p => p.is_default) ?? profiles[0]`

- Call `submitJob(entry.path, profile.config as Record<string, unknown>)`
- On success, call `upsertJob(job)` to add to store
- Optionally show brief success feedback (e.g., flash the row green briefly)

**Zustand rules (MEMORY.md):** Individual selectors only — `useJobsStore(s => s.upsertJob)`, NOT `useJobsStore(s => ({ upsertJob: s.upsertJob, profiles: s.profiles }))`.

**TypeScript rules (MEMORY.md):** Use `import type { BrowseEntry }` for type-only imports.
